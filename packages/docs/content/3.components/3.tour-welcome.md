# TourWelcome

An optional welcome modal that asks users if they want to take the tour.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isVisible` | `boolean` | required | Whether the modal is visible. |
| `title` | `string` | `'Welcome!'` | Modal title. |
| `description` | `string` | `'Would you like a quick guided tour?'` | Main description text. |
| `subdescription` | `string` | `''` | Additional text below description. |
| `logoUrl` | `string` | `''` | URL for a logo image. Hidden if empty. |
| `logoAlt` | `string` | `'Logo'` | Alt text for the logo. |
| `startLabel` | `string` | `'Start'` | Text for the start button. |
| `skipLabel` | `string` | `"No thanks, I'll explore on my own"` | Text for the skip button. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `startTour` | — | User chose to start the tour. |
| `skipTour` | — | User chose to skip. |

## Usage

```vue
<TourWelcome
  :is-visible="showWelcome"
  title="Welcome to MyApp!"
  description="This app helps you manage tasks. Want a quick tour?"
  subdescription="Takes about 2 minutes."
  logo-url="/logo.png"
  logo-alt="MyApp Logo"
  start-label="Let's go!"
  skip-label="Maybe later"
  @start-tour="handleStart"
  @skip-tour="handleSkip"
/>
```

## Features

- Centered modal with backdrop
- Optional logo image
- All text is customizable via props
- Escape key closes the modal
- Focus management
- Dark mode support
