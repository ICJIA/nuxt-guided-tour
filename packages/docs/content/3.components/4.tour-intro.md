# TourIntro

Optional informational slides displayed before the actual tour steps begin. Useful for explaining concepts or features before the hands-on walkthrough.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isVisible` | `boolean` | required | Whether the intro is visible. |
| `slides` | `IntroSlide[]` | `[]` | Array of slide data. Component only renders if slides are provided. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `next` | — | User completed all slides (start the tour). |
| `skip` | — | User chose to skip. |

## Usage

```vue
<script setup>
const introSlides = [
  {
    title: 'What is TaskFlow?',
    icon: 'i-heroicons-sparkles',
    content: ['TaskFlow helps you organize your work efficiently.'],
    benefits: [
      { icon: 'i-heroicons-clock', title: 'Save Time', description: 'Automate repetitive tasks.' },
      { icon: 'i-heroicons-users', title: 'Collaborate', description: 'Work together in real-time.' }
    ],
    footer: 'Used by thousands of teams worldwide.'
  }
]
</script>

<TourIntro
  :is-visible="showIntro"
  :slides="introSlides"
  @next="startTour"
  @skip="skipAll"
/>
```

## Slide Types

Each slide can include different content blocks:

- **`content`**: Array of text paragraphs
- **`examples`**: Grid of syntax examples with labels
- **`benefits`**: List of feature benefits with icons
- **`conversions`**: Before/after comparison cards
- **`callout`**: Highlighted information box
- **`footer`**: Italic footer text

See the [Types](/types) page for the full `IntroSlide` interface.
