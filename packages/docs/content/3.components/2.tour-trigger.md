# TourTrigger

A button to manually start or restart the tour.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | `'Take a tour'` | Button label text. |
| `icon` | `string` | `'i-heroicons-question-mark-circle'` | Button icon. |
| `variant` | `string` | `'ghost'` | Button variant (solid, soft, ghost, outline, link). |
| `color` | `string` | `'neutral'` | Button color. |
| `size` | `string` | `'sm'` | Button size (xs, sm, md, lg, xl). |
| `iconOnly` | `boolean` | `false` | Show only the icon. |
| `tooltip` | `string` | `'Start the guided tour'` | Tooltip text. |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `click` | â€” | Button was clicked. |

## Usage

```vue
<TourTrigger
  label="Help"
  icon="i-heroicons-question-mark-circle"
  variant="soft"
  color="primary"
  @click="restartTour"
/>
```

### Icon-Only Mode

```vue
<TourTrigger
  icon-only
  tooltip="Take a tour"
  @click="restartTour"
/>
```
