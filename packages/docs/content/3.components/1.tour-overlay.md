# TourOverlay

The main tour dialog component. Displays step content with navigation controls.

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `isActive` | `boolean` | Yes | Whether the tour is currently active. |
| `currentStep` | `TourStep \| null` | Yes | Current step data. |
| `progress` | `TourProgress` | Yes | Progress information (current, total, percentage). |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `next` | — | User clicked Next or pressed ArrowRight. |
| `previous` | — | User clicked Back or pressed ArrowLeft. |
| `cancel` | — | User clicked Skip or pressed Escape. |

## Usage

```vue
<TourOverlay
  :is-active="tour.isActive.value"
  :current-step="tour.currentStep.value"
  :progress="tour.progress.value"
  @next="tour.next"
  @previous="tour.previous"
  @cancel="tour.cancel"
/>
```

## Features

- Centered dialog with semi-transparent backdrop
- Keyboard navigation (ArrowRight, ArrowLeft, Escape)
- Focus management (traps focus in dialog)
- Scrollable content area for long descriptions
- Keyboard shortcut display (Mac + Windows)
- Responsive (stacks on mobile)
- Dark mode support
- Reduced motion support
