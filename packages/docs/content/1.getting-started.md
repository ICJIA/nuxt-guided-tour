# Getting Started

## Installation

Install `nuxt-guided-tour` and its peer dependencies:

```bash
npx nuxi module add nuxt-guided-tour
```

Or manually:

```bash
pnpm add nuxt-guided-tour @nuxt/ui @vueuse/core
```

## Configuration

Add the module to your `nuxt.config.ts`:

```typescript
export default defineNuxtConfig({
  modules: ['nuxt-guided-tour']
})
```

## Define Your Tour Steps

Create a tour configuration file (e.g., `tour.config.ts`):

```typescript
import type { TourConfig } from 'nuxt-guided-tour/dist/runtime/types'

export const tourConfig: TourConfig = {
  version: 1,
  autoStart: true,
  autoStartDelay: 800,
  storageKeyPrefix: 'my-app-tour',
  steps: [
    {
      id: 'header',
      target: '[data-tour="header"]',
      title: 'Welcome',
      content: 'This is the header of our application.',
      icon: 'i-heroicons-home',
      position: 'bottom'
    },
    {
      id: 'sidebar',
      target: '[data-tour="sidebar"]',
      title: 'Navigation',
      content: 'Use the sidebar to navigate between sections.',
      icon: 'i-heroicons-bars-3',
      position: 'right'
    }
  ]
}
```

## Mark Your Elements

Add `data-tour` attributes to elements you want to highlight:

```vue
<template>
  <header data-tour="header">My App</header>
  <aside data-tour="sidebar">Navigation</aside>
</template>
```

## Wire Up Components

In your page or layout:

```vue
<script setup lang="ts">
import { tourConfig } from '~/tour.config'

const tour = useTour(tourConfig)

const showWelcome = ref(false)

onMounted(() => {
  if (tour.autoStart && !tour.hasCompletedTour.value) {
    setTimeout(() => {
      showWelcome.value = true
    }, tour.autoStartDelay)
  }
})

function handleStart() {
  showWelcome.value = false
  tour.start()
}

function handleSkip() {
  showWelcome.value = false
  tour.markAsSeen()
}
</script>

<template>
  <!-- Your app content with data-tour attributes -->

  <TourWelcome
    :is-visible="showWelcome"
    title="Welcome!"
    description="Want a quick tour?"
    @start-tour="handleStart"
    @skip-tour="handleSkip"
  />

  <TourOverlay
    :is-active="tour.isActive.value"
    :current-step="tour.currentStep.value"
    :progress="tour.progress.value"
    @next="tour.next"
    @previous="tour.previous"
    @cancel="tour.cancel"
  />

  <TourTrigger @click="tour.resetCompletion(); showWelcome = true" />
</template>
```

That's it! The tour will auto-start for first-time visitors and persist completion in localStorage.
