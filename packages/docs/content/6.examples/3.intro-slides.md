# Intro Slides

Add educational slides before the hands-on tour.

## Flow

1. **Welcome** modal asks if user wants a tour
2. **Intro slides** explain concepts (optional, any number of slides)
3. **Tour steps** walk through the actual UI

## Defining Slides

```typescript
import type { IntroSlide } from 'nuxt-guided-tour/dist/runtime/types'

export const introSlides: IntroSlide[] = [
  {
    title: 'What is TaskFlow?',
    icon: 'i-heroicons-sparkles',
    content: [
      'TaskFlow is a project management tool designed for small teams.',
      'It combines kanban boards, timelines, and team chat in one place.'
    ],
    examples: [
      { label: 'Create task', syntax: '+ New Task', result: 'Task created' },
      { label: 'Assign', syntax: '@username', result: 'Assigned' }
    ],
    footer: 'Used by 10,000+ teams worldwide.'
  },
  {
    title: 'Key Benefits',
    icon: 'i-heroicons-light-bulb',
    content: ['Here\'s why teams love TaskFlow:'],
    benefits: [
      { icon: 'i-heroicons-clock', title: 'Save Time', description: 'Automate repetitive tasks and workflows.' },
      { icon: 'i-heroicons-users', title: 'Real-time Collaboration', description: 'See changes instantly across your team.' },
      { icon: 'i-heroicons-chart-bar', title: 'Analytics', description: 'Track progress with built-in dashboards.' }
    ]
  }
]
```

## Wiring It Up

```vue
<script setup>
import { tourConfig } from '~/tour.config'
import { introSlides } from '~/intro.config'

const tour = useTour(tourConfig)
const showWelcome = ref(false)
const showIntro = ref(false)

onMounted(() => {
  if (tour.autoStart && !tour.hasCompletedTour.value) {
    setTimeout(() => { showWelcome.value = true }, tour.autoStartDelay)
  }
})

function handleWelcomeStart() {
  showWelcome.value = false
  showIntro.value = true  // Show intro slides first
}

function handleIntroComplete() {
  showIntro.value = false
  tour.start()  // Then start the actual tour
}
</script>

<template>
  <TourWelcome :is-visible="showWelcome" @start-tour="handleWelcomeStart" @skip-tour="showWelcome = false; tour.markAsSeen()" />
  <TourIntro :is-visible="showIntro" :slides="introSlides" @next="handleIntroComplete" @skip="showIntro = false; tour.markAsSeen()" />
  <TourOverlay :is-active="tour.isActive.value" :current-step="tour.currentStep.value" :progress="tour.progress.value" @next="tour.next" @previous="tour.previous" @cancel="tour.cancel" />
</template>
```

The `TourIntro` component is entirely optional. If you pass an empty `slides` array, it won't render at all.
