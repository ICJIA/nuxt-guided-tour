# Custom Welcome

Add a welcome modal that asks before starting the tour.

```vue
<script setup>
import { tourConfig } from '~/tour.config'

const tour = useTour(tourConfig)
const showWelcome = ref(false)

onMounted(() => {
  if (tour.autoStart && !tour.hasCompletedTour.value) {
    setTimeout(() => {
      showWelcome.value = true
    }, tour.autoStartDelay)
  }
})

function handleStart() {
  showWelcome.value = false
  tour.start()
}

function handleSkip() {
  showWelcome.value = false
  tour.markAsSeen()
}
</script>

<template>
  <!-- Your app content -->

  <TourWelcome
    :is-visible="showWelcome"
    title="Welcome to TaskFlow!"
    description="TaskFlow helps you manage your projects. Want a quick tour?"
    subdescription="Takes about 1 minute to complete."
    logo-url="/images/logo.svg"
    logo-alt="TaskFlow Logo"
    start-label="Show me around"
    skip-label="I'll figure it out"
    @start-tour="handleStart"
    @skip-tour="handleSkip"
  />

  <TourOverlay
    :is-active="tour.isActive.value"
    :current-step="tour.currentStep.value"
    :progress="tour.progress.value"
    @next="tour.next"
    @previous="tour.previous"
    @cancel="tour.cancel"
  />

  <!-- Restart button -->
  <TourTrigger
    label="Help"
    icon="i-heroicons-question-mark-circle"
    variant="soft"
    color="primary"
    @click="tour.resetCompletion(); showWelcome = true"
  />
</template>
```

All text in `TourWelcome` is customizable via props. The logo is optional â€” leave `logoUrl` empty to hide it.
